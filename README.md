# chaek.org 번역 용어집

학술 번역용 한영 용어집.

## 형식

각 용어집 파일(.tsv)은 탭 구분 형식을 따른다.

```
#en	ko
term	번역어
```

- `#` 이후는 주석 (줄 시작, 줄 중간 모두 가능)
